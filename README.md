# Expense Tracker (Python)

A simple and clean console-based application to track daily expenses.

## Features
- Add expenses with amount, category, and notes
- Automatically records date & time
- View all expenses stored in a text file
- Lightweight and beginner-friendly project

## Technologies Used
- Python
- File Handling (Read/Write)
- Date & Time Module

## How to Run
python expense_tracker.py
